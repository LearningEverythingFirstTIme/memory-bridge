## Key Lessons

### Svelte Class Directive Constraints
- Cannot use Tailwind opacity modifiers (`/20`, `/30`, `/50`, etc.) in Svelte class directives
- `class:bg-umbreon-purple/20` is INVALID — must use `class:bg-umbreon-purple` or inline styles
- Duplicate class directives on same element cause "Attributes need to be unique" errors

### Subagent Quality Issues
- The subagent made multiple syntax errors that required manual fixing
